 🚀 TaskFlow – Smart Daily Planner

Stay organized and boost your productivity with TaskFlow, a sleek and interactive to-do list application built using HTML, CSS, and JavaScript. Whether it's managing daily goals, tracking important tasks,
or celebrating completed work, TaskFlow makes task management simple and enjoyable.

✨ Highlights

* ➕ Quickly add new tasks
* ✏️ Update tasks anytime
* 🗑️ Remove unwanted tasks
* ✅ Move completed tasks to a dedicated section
* 📋 Separate Pending & Completed task lists
* 📅 Track when tasks are added and completed
* 📱 Clean, responsive, and user-friendly design

💡 Designed to help users stay focused, organized, and productive while showcasing dynamic web development concepts such as DOM manipulation, event handling, and interactive UI design.
